# 03 训练模式选择

来源：draw.io 页面 `p03 03 训练模式选择`

阶段：P1 至少支持 100m 射校选择；P2/P3 再开放后续训练模式的完整流程。

## 功能

作为玩家，我希望在模式选择界面查看四类训练的摘要，选择一个模式后确认进入。

```gherkin
# language: zh-CN
功能: 训练模式选择

  背景:
    假如玩家处于“训练模式选择”界面
    而且界面显示 100m精度射校靶、移动目标射击、堑壕射击、城镇攻防四张卡片

  场景: 默认选中 100m 精度射校靶
    当模式选择界面加载完成
    那么“100m精度射校靶”卡片应处于选中状态
    而且“确认进入”按钮应可用

  场景大纲: 选择不同训练卡片
    当玩家选择“<训练模式>”卡片
    那么该卡片应显示选中边框
    而且其他训练卡片应取消选中
    而且当前待确认模式应为“<训练模式>”

    例子:
      | 训练模式 |
      | 100m精度射校靶 |
      | 移动目标射击 |
      | 堑壕射击 |
      | 城镇攻防 |

  场景大纲: 确认进入所选模式
    假如玩家已选择“<训练模式>”
    当玩家点击“确认进入”
    那么系统应进入“<目标界面>”

    例子:
      | 训练模式 | 目标界面 |
      | 100m精度射校靶 | 100m任务说明 |
      | 移动目标射击 | 移动靶设置 |
      | 堑壕射击 | 堑壕地图选择 |
      | 城镇攻防 | 城镇地图选择 |

  场景: 返回主界面
    当玩家点击“返回”
    那么系统应回到“游戏主界面”
    而且不会改变当前已装备武器和已保存设置
```

## Unity 验收点

- 卡片可用 Button、Toggle 或自定义 XR Interactable 实现，但同一时间只能有一个选中状态。
- `确认进入` 不应依赖视觉高亮，应读取真实的 `SelectedTrainingMode`。
