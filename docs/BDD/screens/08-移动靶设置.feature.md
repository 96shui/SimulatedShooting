# 08 移动靶设置

来源：draw.io 页面 `p08 08 移动靶设置`

阶段：P2。

## 功能

作为玩家，我希望在移动目标射击前选择白天或夜晚模式，并选择符合模式的目标速度。

```gherkin
# language: zh-CN
功能: 移动目标射击模式设置

  背景:
    假如玩家选择了移动目标射击
    那么系统应显示“移动目标射击：模式设置”

  场景: 默认选择白天模式
    当移动靶设置界面加载完成
    那么“白天模式”应处于选中状态
    而且速度选择应显示 3m/s、4m/s、5m/s
    而且默认速度应为 4m/s 或项目配置的默认值

  场景: 切换到夜晚模式
    当玩家选择“夜晚模式”
    那么“夜晚模式”应高亮
    而且速度选择应显示 2m/s、2.5m/s、3m/s
    而且右侧预览应显示夜晚训练将使用微光瞄准镜

  场景大纲: 选择目标速度
    假如玩家选择了“<模式>”
    当玩家选择速度“<速度>”
    那么当前移动靶速度应记录为“<速度>”
    而且开始训练时应使用该速度

    例子:
      | 模式 | 速度 |
      | 白天模式 | 3m/s |
      | 白天模式 | 4m/s |
      | 白天模式 | 5m/s |
      | 夜晚模式 | 2m/s |
      | 夜晚模式 | 2.5m/s |
      | 夜晚模式 | 3m/s |

  场景: 开始训练
    假如玩家已选择昼夜模式和目标速度
    当玩家点击“开始训练”
    那么系统应创建移动目标训练 Session
    而且应进入对应的移动靶 HUD
    而且训练开始前应进入 3 秒等待状态

  场景: 返回上一级
    当玩家点击“返回”
    那么系统应返回主界面或训练模式选择
    而且不应创建移动目标训练 Session
```

## Unity 验收点

- 昼夜模式和速度选择建议使用同一配置对象传给移动靶状态机。
- 夜晚模式下不允许选择白天专属速度，反之亦然。
