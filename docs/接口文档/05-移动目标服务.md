# 05 移动目标服务

## 目标

约束移动目标射击的昼夜模式、速度选择、3 秒等待、目标路线、端点禁射、点射记录和结算评级。

## 业务规则

- 白天速度只能为 3m/s、4m/s、5m/s。
- 夜晚速度只能为 2m/s、2.5m/s、3m/s。
- 开始后先等待 3 秒，等待期间 `CanShoot == false`。
- 目标先右到左，再左到右。
- 左端点停留 2 秒，停留期间 `CanShoot == false`。
- 总弹药 10 发，每次点射消耗 2 发，最多 5 次点射。
- 抵达右侧终点后自动结算。
- 命中 3 发及格，4 发良好，5 发及以上优秀。

## 错误处理

| 条件 | 错误码 |
|---|---|
| 模式和速度不匹配 | InvalidInput |
| 禁射状态记录点射 | InvalidState |
| 弹药不足 2 发仍点射 | InvalidState |
| 超过 5 次点射 | InvalidState |
| 未完成路线请求结算 | InvalidState |

## UI 绑定

- 设置页使用 `GetAvailableSpeeds` 刷新速度按钮。
- HUD 的“可射击/禁止射击”只读取 `CanShoot`。
- 结算页只读取 `MovingTargetResultDto`。
