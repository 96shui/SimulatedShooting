# 01 页面导航与 UI 事件

## 目标

约束所有 draw.io 页面对应的 Screen ID、按钮事件和页面跳转结果，保证 UI 与后端路由状态一致。

## 路由接口

```csharp
public interface IUIRouter
{
    ScreenId Current { get; }
    ServiceResult<ScreenId> Open(ScreenId screen, NavigationArgs args = default);
    ServiceResult<ScreenId> Back();
    bool IsTransitioning { get; }
}

public readonly struct NavigationArgs
{
    public TrainingMode? Mode { get; init; }
    public string SessionId { get; init; }
    public string ReturnToScreen { get; init; }
}
```

## 页面跳转表

| 来源页面 | UI 事件 | 后端命令 | 目标页面 |
|---|---|---|---|
| MainMenu | MainMenu_OpenZeroing | `SetMode(Zeroing)` | ZeroingBriefing |
| MainMenu | MainMenu_OpenMovingTarget | `SetMode(MovingTarget)` | MovingTargetSettings |
| MainMenu | MainMenu_OpenTrench | `SetMode(Trench)` | TrenchMapSelection |
| MainMenu | MainMenu_OpenUrban | `SetMode(Urban)` | UrbanMapSelection |
| MainMenu | MainMenu_OpenArmory | `OpenArmory(returnTo=MainMenu)` | Armory |
| MainMenu | MainMenu_OpenSettings | `OpenSettings(returnTo=MainMenu)` | Settings |
| TrainingModeSelection | Mode_Confirm | `SetMode(selected)` | 对应模式入口 |
| ZeroingBriefing | Zeroing_Start | `ZeroingService.StartSession()` | ZeroingHud |
| ZeroingImpactAnalysis | Zeroing_ApplyAdjustment | `ZeroingService.ApplyAdjustment()` | 当前页 |
| ZeroingImpactAnalysis | Zeroing_NextRound | `ZeroingService.Continue()` | ZeroingHud 或 ZeroingFinalRating |
| MovingTargetSettings | MovingTarget_Start | `MovingTargetService.StartSession()` | MovingTargetDayHud 或 MovingTargetNightHud |
| TrenchMapSelection | Trench_SelectMap | `TrenchService.SelectMap()` | Armory 或 TrenchBriefing |
| Armory | Armory_Equip | `WeaponService.Equip()` | 当前页或 ReturnToScreen |
| TrenchBriefing | Trench_Start | `TrenchService.StartSession()` | TrenchHud |
| TrenchHud | Squad_OpenRadial | `SquadService.OpenCommandMenu()` | SquadCommandRadial |
| UrbanMapSelection | Urban_SelectMap | `UrbanService.SelectMap()` | Armory 或 UrbanStreetHud |
| UrbanStreetHud | Urban_EnterBuilding | `UrbanService.EnterBuilding()` | UrbanBuildingHud |
| Settings | Common_Apply | `SettingsService.Save()` | 当前页 |
| Settings | Common_Back | `SettingsService.DiscardPending()` | ReturnToScreen |

## 路由约束

- `IsTransitioning == true` 时，UI 必须禁用所有会导致切页的按钮。
- `Back()` 必须返回上一业务页面，而不是固定返回主菜单。
- 从训练中进入设置时，返回后不得销毁当前训练 Session。
- 打开环形菜单属于叠层 UI，关闭后应回到原 HUD。

## 事件传递

```csharp
public readonly struct UICommand
{
    public UIEventId EventId { get; init; }
    public ScreenId SourceScreen { get; init; }
    public string PayloadJson { get; init; }
    public double ClientTime { get; init; }
}
```

UI 只发送事件和参数，不直接调用场景物体；Presenter 或 Controller 将事件转成服务命令。
