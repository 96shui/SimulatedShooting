# 06 武器与弹药服务

## 目标

约束武器库、装备、武器属性、弹药显示、换弹和射击消耗接口。

## 阶段范围

| 阶段 | 武器与弹药范围 |
|---|---|
| P1 | 训练用主武器原型、双手持枪、扳机开火、按钮换弹、左右肩切换、简单后坐力、基础弹道或射线命中、100m 射校弹药/HUD 刷新 |
| P2 | 武器库原型、训练模式可用武器筛选、移动靶点射消耗、弹药/HUD 共用服务 |
| P3 | 堑壕/城镇武器装备、战斗弹药、命中敌人反馈 |
| Later | 真实弹匣插拔、拉机柄、保险、卡弹、复杂枪械拆装、细分枪械动画 |
 
P1 不要求真实弹匣插拔、拉机柄或保险机制；这些行为不得阻塞 100m 射校闭环交付。

## DTO

```csharp
public readonly struct WeaponDefinitionDto
{
    public string WeaponId { get; init; }
    public string DisplayName { get; init; }
    public WeaponType Type { get; init; }
    public int MagazineCapacity { get; init; }
    public int MaxReserveAmmo { get; init; }
    public RecoilLevel Recoil { get; init; }
    public IReadOnlyList<TrainingMode> ApplicableModes { get; init; }
}

public enum WeaponType
{
    AssaultRifle,
    Shotgun,
    LightMachineGun
}

public enum RecoilLevel
{
    Low,
    Medium,
    High
}
```

## 服务接口

```csharp
public interface IWeaponService
{
    ServiceResult<IReadOnlyList<WeaponDefinitionDto>> GetWeapons();
    ServiceResult<WeaponDefinitionDto> GetWeapon(string weaponId);
    ServiceResult<WeaponDefinitionDto> GetEquippedWeapon();
    ServiceResult<WeaponDefinitionDto> SelectPreview(string weaponId);
    ServiceResult<WeaponDefinitionDto> Equip(string weaponId, TrainingMode? mode);
}

public interface IAmmoService
{
    ServiceResult<AmmoDto> GetAmmo(string sessionId);
    ServiceResult<AmmoDto> ConsumeAmmo(string sessionId, int amount);
    ServiceResult<AmmoDto> StartReload(string sessionId);
    ServiceResult<AmmoDto> CompleteReload(string sessionId);
}
```

P1 武器控制可以先由输入适配层下发抽象命令，例如开火、换弹、切换肩侧；服务层负责弹药、可射击状态和结果数据，Unity 场景层负责持枪、后坐力和命中表现。

## 业务规则

- 武器库选择武器只更新预览，不改变已装备武器。
- 点击“装备”后才改变当前武器。
- 进入堑壕或城镇任务时读取已装备武器。
- 弹药显示格式为当前弹匣/备用弹药，例如 30/120。
- 玩家自行换弹；换弹开始时可通知队友服务进入掩护状态。
- 弹药不足时不能生成有效射击。
- P1 按钮换弹可以直接完成或经过短暂换弹状态，但必须通过 `StartReload` / `CompleteReload` 或等效命令刷新弹药 DTO。
- P1 左右肩切换影响武器射击参考点和 HUD 提示；不改变弹药规则。
- 禁止射击状态必须在消耗弹药前判断。
- 真实弹匣插拔、拉机柄和保险属于 Later，不进入 P1/P2 验收。

## 事件

| 事件 | 数据 | 触发 |
|---|---|---|
| WeaponChangedEvent | WeaponDefinitionDto | 装备成功 |
| AmmoChangedEvent | AmmoDto | 射击、换弹开始、换弹完成 |
| ReloadStartedEvent | sessionId | 玩家开始换弹 |
| ReloadCompletedEvent | sessionId | 换弹完成 |

## 错误处理

| 条件 | 错误码 |
|---|---|
| 武器不存在 | NotFound |
| 武器不适用于当前模式 | InvalidInput |
| 当前弹药不足 | InvalidState |
| 已在换弹中再次换弹 | Busy |
