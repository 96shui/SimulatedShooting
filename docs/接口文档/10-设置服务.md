# 10 设置服务

## 目标

约束设置界面与 Unity 后端设置服务之间的数据交互，包括临时修改、应用、恢复默认和持久化。

## 相关 BDD

- `22-设置界面.feature.md`

## DTO

```csharp
public readonly struct GameSettingsDto
{
    public ComfortLevel ComfortLevel { get; init; }
    public TurnMode TurnMode { get; init; }
    public MoveMode MoveMode { get; init; }
    public int Brightness { get; init; }
    public int HudOpacity { get; init; }
    public int SfxVolume { get; init; }
    public bool AimAssistEnabled { get; init; }
    public Language Language { get; init; }
}

public enum ComfortLevel
{
    Low,
    Medium,
    High
}

public enum TurnMode
{
    Smooth,
    Snap
}

public enum MoveMode
{
    Teleport,
    Continuous
}

public enum Language
{
    Chinese
}
```

## 服务接口

```csharp
public interface ISettingsService
{
    ServiceResult<GameSettingsDto> Load();
    ServiceResult<GameSettingsDto> GetPending();
    ServiceResult<GameSettingsDto> SetPending(GameSettingsDto settings);
    ServiceResult<GameSettingsDto> ResetPendingToDefault();
    ServiceResult<GameSettingsDto> SavePending();
    ServiceResult<GameSettingsDto> DiscardPending();
    ServiceResult<GameSettingsDto> GetDefault();
}
```

## 默认值

| 字段 | 默认值 |
|---|---|
| ComfortLevel | Medium |
| TurnMode | Smooth |
| MoveMode | Teleport |
| Brightness | 70 |
| HudOpacity | 80 |
| SfxVolume | 80 |
| AimAssistEnabled | true |
| Language | Chinese |

## 数值范围

| 字段 | 范围 |
|---|---|
| Brightness | 0-100 |
| HudOpacity | 0-100 |
| SfxVolume | 0-100 |

## 业务规则

- 打开设置界面时，UI 获取 `Load()` 并拷贝为 Pending 设置。
- 玩家修改设置时调用 `SetPending()`，实时预览读取 Pending。
- 点击“应用”时调用 `SavePending()`，写入持久化并发布 `SettingsChangedEvent`。
- 点击“恢复默认”只修改 Pending，不立即持久化。
- 点击“返回”且未应用时调用 `DiscardPending()`。
- 从训练中打开设置，返回后不得销毁训练 Session。

## Unity 映射

| 设置项 | Unity 影响 |
|---|---|
| TurnMode.Smooth | 启用连续转向 Provider |
| TurnMode.Snap | 启用分段转向 Provider |
| MoveMode.Teleport | 启用 Teleportation Ray |
| MoveMode.Continuous | 启用连续移动 Provider |
| HudOpacity | 修改 HUD CanvasGroup alpha |
| SfxVolume | 修改 AudioMixer 音效组 |
| Brightness | 修改后处理或全局曝光参数 |

## 错误处理

| 条件 | 错误码 |
|---|---|
| 数值超出范围 | InvalidInput |
| 保存失败 | PersistenceFailed |
| 设置文件损坏 | PersistenceFailed，并回退默认值 |
