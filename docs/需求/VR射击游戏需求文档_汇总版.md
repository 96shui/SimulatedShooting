# VR射击游戏需求文档

---

## 一、100m精度射校靶

| 项目 | 内容 |
|------|------|
| 射击距离 | 100m |
| 射击模式 | 单发射，每轮3发，共3轮 |
| 靶标 | 标准胸靶 50cm×50cm，10环为直径10cm白圈 |
| 偏移预设 | 开局随机生成一次弹着点偏移（在靶内但10环外），3轮共用同一偏移量 |
| 弹着点散布 | 3发弹着点由玩家射击晃动决定，非随机散布；准星柱和觇孔随枪晃动，玩家需稳定据枪 |
| 平正关系 | 准星柱与觇孔存在平正归零关系，两者相对位置决定弹着点；瞄具随枪体同步晃动 |
| 垂直调整 | 偏上→准星柱逆时针调X°，偏下→顺时针调X°（23cm一周，0.064cm≈1°） |
| 水平调整 | 偏左→觇孔向前调X格，偏右→向后调X格（一格2cm，向上取整） |
| 每轮弹窗 | 显示垂直/水平偏差，提示准星柱旋转方向和角度、觇孔调整方向和格数 |
| 通过条件 | 调整后3发全部落在10环内（距中心≤5cm） |
| 评级 | 第1轮通过=优秀，第2轮=良好，第3轮=及格，未通过=不及格 |

## 二、移动目标射击

| 项目 | 内容 |
|------|------|
| 射击距离 | 100m |
| 射击模式 | 点射，一次出两发，共10发弹（5次点射） |
| 靶标 | 侧身跑步靶 |
| 移动范围 | 水平40m，左右各20m |
| 运动方向 | 先右到左，再左到右 |
| 端点标识 | 左右两端点设置标识（如地图插旗） |
| 端点停留 | 运动到左端（左侧20m）后停2s，停留期间禁止射击，再反向运动；抵达右侧端点结束 |
| 运动次数 | 来回运动两次（右→左→右） |
| 瞄具 | 白天用觇孔，夜晚用微光瞄准镜 |
| 模式选择 | 第一步选昼夜：白天模式/夜晚模式；第二步选速度：白天可选3m/s、4m/s、5m/s，夜晚可选2m/s、2.5m/s、3m/s |
| 开始流程 | 选完模式→切到射击视角→等待3s→靶子开始运动→方可射击 |
| 结束条件 | 靶子抵达右侧端点，给出评价 |
| 命中评级 | 3发及格，4发良好，5发及以上优秀 |

## 三、堑壕射击

| 项目 | 内容 |
|------|------|
| 地图数量 | 2张 |
| 视角切换 | 第一人称/第三人称可切换 |
| 地图显示 | 小地图显示当前位置、队友位置、敌人标红位置 |
| 武器选择 | 19-1自动步枪、95-1自动步枪、QBS-09霰弹枪、QJB-201班用机枪 |
| 武器库 | 武器墙，对着可打开武器库选枪 |
| 开场流程 | 选枪→来到堑壕边→无人机起飞动画→显示堑壕平面投影图→进入堑壕 |
| 姿态控制 | 卧倒、蹲下、起立 |
| 小队配置 | 3人纵队，玩家在最前面朝前，第二名朝前在玩家后面，最后一名朝后 |
| 弹药总量 | 150发，弹匣30发，最多持有5个弹匣（枪上1个+身上4个） |
| 弹量显示 | 当前弹匣/备用弹匣，如30/120 |
| 换弹机制 | 玩家自行换弹，换弹时第二名队友自动上前掩护 |
| 换肩射击 | 支持左右肩切换射击 |
| 拐角射击 | 拐角处可将枪伸出去射击 |
| 敌人设置 | 开局生成3-5个，平面图上标红位置，实际出现位置在标红位置附近随机偏移（不能离标红位置太远） |
| 敌人出现 | 堑壕内拐角随机出现 |
| 敌人尸体 | 死亡后尸体不消失 |
| 敌人中弹 | 射击命中敌人有中弹音效 |
| 遇敌机制 | 拐角遇敌射击时，后方队友可将玩家往后拉 |
| 队友指令 | 可指令后方队友丢手榴弹、火力支援 |
| 胜利条件 | 堑壕搜索完且消灭全部敌人 |
| 失败条件 | 玩家死亡 |
| 道具 | 敌人手榴弹（可拾取/规避） |
