# task005 基础武器控制：持枪、开火、换弹、切肩

## 负责人

功能B-武器输入与AI基础

## 目标

完成 P1 训练用基础武器控制，让玩家可以双手持枪、扳机开火、按钮换弹、左右肩切换，并能向玩法服务输出射击结果。

## 交付内容

- 训练用主武器原型。
- 双手持枪基础交互：
  - 无 VR 时可用测试/调试输入模拟。
  - 有 VR 时预留 XR Grab / XR Interactor 接入点。
- 扳机开火。
- 按钮换弹。
- 左右肩切换。
- 简单后坐力表现。
- 基础弹道或射线命中。
- 当前弹匣、备用弹药、可射击状态输出。

## 不包含

- 不做真实弹匣插拔。
- 不做拉机柄、保险、卡弹。
- 不做多武器完整切换。

## 依赖关系

- 前置依赖：task003、task004。
- 后续依赖：task009、task013、task014。

## 联调说明

- 与 功能A 联调：开火结果、弹药变化、肩侧状态要传入 100m 射校服务或 HUD 服务。
- 与 场景 联调：确认射线/弹道可以命中靶标。
- 与 UI 联调：换弹和左右肩状态能显示。

## 测试要求

- EditMode 单元测试：
  - 弹药消耗。
  - 弹药不足时不能开火。
  - 换弹后弹匣和备用弹药正确。
  - 左右肩切换状态正确。
- PlayMode 测试：
  - 输入替身触发开火后产生射击结果。
  - 输入替身触发换弹和切肩后 HUD 可收到变化。

## 验收标准

- 无 VR 设备时可以完成开火、换弹、切肩。
- 开火前检查 `CanShoot` 和弹药，不允许先扣弹再回滚。
- 武器状态能被 P1 HUD 和 100m 服务消费。
- EditMode 测试通过。
