## 业务规则

- 每局最多 3 轮，每轮 3 发。
- 开局生成一次固定弹着偏移，3 轮共用。
- 10 环判定为距中心 `<= 5cm`。
- 垂直调整：偏上提示准星柱逆时针，偏下提示顺时针。
- 水平调整：偏左提示觇孔向前，偏右提示向后。
- 水平一格 2cm，向上取整。
- 23cm 一周，0.064cm 约等于 1 度。
- 第一轮通过为优秀，第二轮良好，第三轮及格，未通过不及格。

## 错误处理

| 条件 | 错误码 |
|---|---|
| Session 不存在 | NotFound |
| 弹数为 0 仍记录射击 | InvalidState |
| 一轮未满 3 发请求分析 | InvalidState |
| 同一轮重复应用调整 | None，返回原分析结果 |
| 第 3 轮后请求下一轮 | InvalidState 或返回最终结果 |

## UI 绑定

- `ZeroingHud` 只读取 `ZeroingSessionDto` 和 `ZeroingShotDto`。
- `ImpactAnalysis` 只读取 `ZeroingRoundAnalysisDto`。
- `FinalRating` 只读取 `ZeroingResultDto`。
