## 枚举和 DTO

```csharp
public enum SquadMemberRole
{
    Player,
    TeammateTwo,
    TeammateThree
}

public enum SquadMemberState
{
    Normal,
    Following,
    CoveringReload,
    SuppressingFire,
    ThrowingGrenade,
    MovingForward,
    HoldingPosition,
    Down
}

public enum SquadCommandType
{
    Grenade,
    FireSupport,
    CoverReload,
    MoveForward,
    Stop
}

public readonly struct SquadMemberDto
{
    public string MemberId { get; init; }
    public SquadMemberRole Role { get; init; }
    public SquadMemberState State { get; init; }
    public Vector3 WorldPosition { get; init; }
    public float Health { get; init; }
}

public readonly struct SquadStatusDto
{
    public IReadOnlyList<SquadMemberDto> Members { get; init; }
    public bool CommandMenuAvailable { get; init; }
}
```

## 指令接口

```csharp
public readonly struct SquadCommandRequest
{
    public string SessionId { get; init; }
    public SquadCommandType CommandType { get; init; }
    public string TargetAreaId { get; init; }
    public Vector3? TargetWorldPosition { get; init; }
}

public readonly struct SquadCommandResult
{
    public string CommandId { get; init; }
    public SquadCommandType CommandType { get; init; }
    public bool Accepted { get; init; }
    public string RejectReason { get; init; }
    public SquadStatusDto SquadStatus { get; init; }
}

public interface ISquadCommandService
{
    ServiceResult<SquadStatusDto> GetSquadStatus(string sessionId);
    ServiceResult<IReadOnlyList<SquadCommandType>> GetAvailableCommands(string sessionId);
    ServiceResult<SquadCommandResult> Issue(SquadCommandRequest request);
    ServiceResult<SquadStatusDto> OnReloadStarted(string sessionId);
}
```

## 业务规则

- 小队默认三人纵队：玩家最前，二号队友在后方朝前，三号队友最后朝后。
- 玩家换弹时二号队友自动进入掩护换弹状态。
- 手榴弹指令必须有有效敌情区域或目标位置。
- 火力支援指令需要敌人或敌情预估方向。
- 停止指令使队友停止前进并保持当前位置。
- 指令失败必须返回原因，UI 不应假装执行成功。

## 事件

| 事件 | 触发 | 用途 |
|---|---|---|
| SquadStatusChangedEvent | 队友状态变化 | HUD 刷新队友状态 |
| SquadCommandIssuedEvent | 指令接受 | AI 执行动作 |
| SquadCommandRejectedEvent | 指令拒绝 | UI 显示失败原因 |
| SquadCoverReloadEvent | 玩家换弹 | 二号队友掩护 |

## 错误处理

| 条件 | 错误码 |
|---|---|
| 当前场景不支持队友指令 | InvalidState |
| 没有可用队友 | InvalidState |
| 手榴弹没有目标 | InvalidInput |
| 指令冷却中 | Cooldown |
| AI 正在执行不可打断动作 | Busy |
