# 10 移动靶夜晚HUD

来源：draw.io 页面 `p10 10 移动靶夜晚HUD`

## 功能

作为玩家，我希望在夜晚移动靶 HUD 中通过微光瞄准镜射击，并遵守端点停留禁止射击规则。

```gherkin
# language: zh-CN
功能: 移动目标夜晚 HUD

  背景:
    假如玩家以夜晚模式开始移动目标射击
    而且目标速度为 2.5m/s
    那么系统应显示夜间靶场和微光瞄准镜视野

  场景: 夜晚 HUD 使用微光镜显示
    当夜晚 HUD 加载完成
    那么中心视野应显示绿色微光镜效果
    而且 HUD 文本仍应保持清晰可读
    而且弹药、点射模式和命中数应显示在固定位置

  场景: 显示夜晚移动状态
    当目标从左向右移动
    那么 HUD 应显示目标速度 2.5m/s
    而且应显示方向左→右
    而且路线区域应显示端点标识或发光插旗

  场景: 端点停留期间禁止射击
    假如目标处于端点停留状态
    当玩家扣动扳机
    那么 HUD 应显示“禁止射击”或等效提示
    而且弹药数量不应减少
    而且命中数不应变化

  场景: 夜晚模式命中统计
    假如目标处于可射击状态
    当玩家使用微光镜完成有效点射并命中目标
    那么命中数应增加
    而且点射节奏条应记录本次点射结果

  场景: 夜晚训练结束
    假如目标完成规定路线
    当目标抵达右侧终点
    那么系统应进入移动目标训练结算
    而且结算数据应记录夜晚模式和 2.5m/s 速度
```

## Unity 验收点

- 夜晚视觉效果可以是后处理、材质或镜头 UI，但 BDD 验收关注可读性和状态正确性。
- 禁射状态必须在武器发射逻辑前拦截，不能先扣弹再回滚。
