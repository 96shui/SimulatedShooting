# 04 100m任务说明

来源：draw.io 页面 `p04 04 100m任务说明`

## 功能

作为玩家，我希望在开始 100m 精度射校靶前确认训练规则和通过条件。

```gherkin
# language: zh-CN
功能: 100m 精度射校靶任务说明

  背景:
    假如玩家选择了 100m 精度射校靶
    那么系统应显示 100m 任务说明界面

  场景: 展示任务参数
    当任务说明界面加载完成
    那么界面应显示射击距离为 100m
    而且射击模式为单发射
    而且显示每轮 3 发、共 3 轮
    而且显示目标为 50cm×50cm 胸靶
    而且显示 10 环直径为 10cm
    而且显示通过条件为 3 发全部进入 10 环

  场景: 显示胸靶示意
    当玩家查看任务说明界面
    那么右侧应显示胸靶示意和中心 10 环
    而且示意图只作为说明，不应消耗射击输入

  场景: 开始射击
    当玩家点击“开始射击”
    那么系统应创建 100m 射校训练 Session
    而且应生成本局固定弹着偏移
    而且系统应进入“100m射击HUD”
    而且轮次应为 1/3，弹数应为 3/3

  场景: 返回模式选择
    当玩家点击“返回”
    那么系统应返回“训练模式选择”或上一级模式入口
    而且不应创建 100m 射校训练 Session
```

## Unity 验收点

- 开始训练时初始化 `ZeroingSession`，包含轮次、弹药、固定偏移、弹着记录和调整状态。
- 固定偏移应可注入随机种子，便于自动化测试复现。
